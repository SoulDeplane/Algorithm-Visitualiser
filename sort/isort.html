Insertion Sort

Description:

Insertion Sort builds a sorted array one element at a time by inserting each element into its correct position. It works similarly to how you might sort playing cards in your handâ€”by taking one card at a time and placing it in the correct order among the others.

#include<iostream>
using namespace std;
void isort(int arr[], int n)
{
    	for (int i = 1; i < n; i++)
	{
        	int key = arr[i];
        	int j = i - 1;
        	while (j >= 0 && arr[j] > key)
		{
            		arr[j + 1] = arr[j];
            		j--;
        	}
        	arr[j + 1] = key;
    	}
}

def isort(arr):
    	n = len(arr)
    	for i in range(1, n):
        	key = arr[i]
        	j = i - 1        
        	while j >= 0 and arr[j] > key:
            		arr[j + 1] = arr[j]
            		j -= 1
        	arr[j + 1] = key

arr = eval(input("Enter list: "))

Time Complexity:

  • Best Case: O[n] (when the list is already sorted, only one pass is needed)
  • Worst Case: O[n2] (when the list is sorted in reverse order, requiring multiple shifts and insertions)
  • Average Case: O[n2]

Space Complexity:

  • O[1] (in-place, does not require additional space)

Used for:

  • Efficient for small or nearly sorted datasets.

Applications:

  • Online Sorting: When elements arrive sequentially and need to be sorted dynamically.
  • Nearly Sorted Data: Performs well when minimal reordering is needed.
  • Educational Use: A simple and intuitive sorting method that aids learning.
