Breadth-First Search (BFS)

☰ Home | Searching Algorithms | Sorting Algorithms | Graph Algorithms | Greedy Algorithms | Code Complexity Analyzer

Description:

BFS explores a graph level-by-level using a queue, ensuring all nodes at the same depth are visited before moving deeper. It finds the shortest path in unweighted graphs.

#include<iostream>
#include<vector>
#include<queue>
using namespace std;
void bfs(vector>& graph, int start)
{
    	vector visited(graph.size(), false);
    	queue q;
    	q.push(start);
    	visited[start] = true;
    	cout << "BFS Traversal: ";
    	while (!q.empty())
	{
    	    	int node = q.front();
    	    	q.pop();
    	    	cout << node << " ";
    	    	for (int neighbor : graph[node])
		{
    	    	    	if (!visited[neighbor])
			{
    	    	    	    	q.push(neighbor);
    	    	    	    	visited[neighbor] = true;
    	    	    	}
    	    	}
    	}
    	cout << endl;
}

from collections import deque

def bfs(graph, start):
    	visited = set()
    	queue = deque([start])
    	print("BFS Traversal:", end=" ")
    	while queue:
    	    	node = queue.popleft()
    	    	if node not in visited:
    	    	    	print(node, end=" ")
    	    	    	visited.add(node)
    	    	    	queue.extend(graph[node] - visited)
    	print()

graph = {}
nodes, edges = input("Enter number of nodes and edges: ").split()
nodes = int(nodes)
edges = int(edges)
print("Enter edges (node1 node2):")
for _ in range(edges):
    	u, v = map(int, input().split())
    	graph.setdefault(u, set()).add(v)
    	graph.setdefault(v, set()).add(u) 
start = int(input("Enter starting node: "))
bfs(graph, start)

Time Complexity:

  • Best Case: O[V+E] (where V is vertices and E is edges)
  • Worst Case: O[V+E]
  • Average Case: O[V+E]

Space Complexity:

  • O[V] (due to queue storage)

Used for:

  • Shortest path finding and traversal of graphs.

Applications:

  • Social Network Analysis: Finding connections between users.
  • Shortest Path in Unweighted Graphs: Optimal for game AI pathfinding.
