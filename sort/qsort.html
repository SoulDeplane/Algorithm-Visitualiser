Quick Sort

Description:

Quick Sort is a divide-and-conquer algorithm that selects a pivot element, partitions the array into elements smaller and larger than the pivot, and recursively sorts both partitions.

Code Implementation: 


#include<iostream>
using namespace std;
int partition(int arr[], int left, int right)
{
    	int pivot = arr[right];
    	int i = left - 1;
    	for (int j = left; j < right; j++)
	{
        	if (arr[j] < pivot)
		{
        	    	i++;
        	    	swap(arr[i], arr[j]);
        	}
    	}
    	swap(arr[i + 1], arr[right]);
    	return i + 1;
}

void qsort(int arr[], int left, int right)
{
    	if (left < right)
	{
        	int p = partition(arr, left, right);
        	qsort(arr, left, p - 1);
        	qsort(arr, p + 1, right);
    	}
}

def partition(arr, left, right):
    	pivot = arr[right]
    	i = left - 1
    	for j in range(left, right):
    	    	if arr[j] < pivot:
    	    	    	i += 1
    	    	    	arr[i], arr[j] = arr[j], arr[i]
    	arr[i + 1], arr[right] = arr[right], arr[i + 1]
    	return i + 1

def qsort(arr, left, right):
    if left < right:
        p = partition(arr, left, right)
        qsort(arr, left, p - 1)
        qsort(arr, p + 1, right)

Time Complexity:

  • Best Case: O[nlogn] (when partitions are balanced)
  • Worst Case: O[n2] (when partitions are highly unbalanced)
  • Average Case: O[nlogn]

Space Complexity:

  • O[nlogn] (due to recursion)

Used for:

  • Fast sorting of large datasets.

Applications:

  • Efficient General-Purpose Sorting: Used in various programming libraries.

  • Databases and File Systems: Used for indexing and organizing data efficiently.
