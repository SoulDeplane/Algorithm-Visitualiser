Count Sort

☰ Home | Searching Algorithms | Sorting Algorithms | Graph Algorithms | Greedy Algorithms

Description:

Counting Sort is a non-comparative sorting algorithm that counts occurrences of each unique value in the dataset and uses this count to place elements in their correct positions.

#include<iostream>
using namespace std;
void csort(int arr[], int n)
{
    	int m = arr[0];
    	for (int i = 1; i < n; i++)
	{
        	if (arr[i] > m)
            		m = arr[i];
        }
    	int co[m + 1] = {0};
    	for (int i = 0; i < n; i++)
    	    	co[arr[i]]++;
    	int in = 0;
    	for (int i = 0; i <= m; i++)
	{
    	    	for (int j = 0; j < co[i]; j++)
		{
    	        	arr[in] = i;
    	        	in++;
    	    	}
    	}
}

Time Complexity:

  • Best Case: O[n+k]
  • Worst Case: O[n+k]
  • Average Case: O[n+k]

Space Complexity:

  • O[k] (additional storage needed for counting array)

Used for:

  • Sorting integers or elements with a limited range.

Applications:

  • Integer Sorting: Ideal when elements have a known range.
  • Linear-Time Sorting Needs: Faster than comparison-based sorting for small ranges.
  • Real-Time Systems: Offers consistent performance characteristics.
