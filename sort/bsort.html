Bubble Sort

☰ Home | Searching Algorithms | Sorting Algorithms | Graph Algorithms | Greedy Algorithms

Description:

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. This process repeats until the list is sorted. After each full pass, the largest element "bubbles" up to its correct position.

#include<iostream>
using namespace std;
void bsort(int arr[], int n)
{
    	for (int i = 0; i < n - 1; i++)
	{
        	for (int j = 0; j < n - i - 1; j++)
		{
            		if (arr[j] > arr[j + 1])
				swap(arr[j], arr[j + 1]);
        	}
    	}
}

def bsort(arr):
    	n = len(arr)
    	for i in range(n - 1):
        	for j in range(n - i - 1):
            		if arr[j] > arr[j + 1]:
                		arr[j], arr[j + 1] = arr[j + 1], arr[j]

arr = eval(input("Enter list: "))

Time Complexity:

  • Best Case: O[n] (when the list is already sorted, only one pass is needed)
  • Worst Case: O[n2] (when the list is sorted in reverse order, requiring multiple passes and swaps)
  • Average Case: O[n2]

Space Complexity:

  • O[1] (in-place, does not require additional space)

Used for:

  • Simple sorting scenarios, teaching sorting algorithms, and cases where data is nearly sorted.

Applications:

  • Educational Use: Frequently used for teaching due to its simplicity and intuitive approach.
  • Nearly Sorted Data: Performs well when data is almost sorted, as minimal swaps are required.
  • Low-Memory Environments: Since it sorts in-place, it is useful when additional storage is constrained.
