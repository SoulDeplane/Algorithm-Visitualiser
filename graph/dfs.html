Depth-First Search (DFS)

☰ Home | Searching Algorithms | Sorting Algorithms | Graph Algorithms | Greedy Algorithms

Description:

DFS explores a graph deeply by visiting nodes recursively and backtracking when needed. It uses a stack (or recursion) to keep track of visited nodes.

Code Implementation:

Time Complexity:

  • Best Case: O[V+E] (where V is vertices and E is edges)
  • Worst Case: O[V+E]
  • Average Case: O[V+E]

Space Complexity:

  • O[V] (due to recusrion or stack storage)

Used for:

  • Graph traversal, cycle detection, and topological sorting.

Applications:

  • Pathfinding in Mazes and Games: Finding possible routes or solutions.
  • Connected Components Detection: Identifying clusters in graphs.
