Selection Sort

☰ Home | Searching Algorithms | Sorting Algorithms | Graph Algorithms | Greedy Algorithms

Description:

Selection Sort repeatedly finds the smallest element in the unsorted portion of the list and swaps it with the first unsorted element. This continues until the entire list is sorted.

Code Implementation: 


#include<iostream>
using namespace std;
void ssort(int arr[], int n)
{
    	for (int i = 0; i < n - 1; i++)
	{
		m=i;
        	for (int j = i + 1; j < n; j++)
		{
            		if (arr[j] > arr[m])
				m = j;
        	}
		swap(arr[i], arr[m]);
    	}
}

main()
{
    	int n;
    	cout << "Enter size of array: ";
    	cin >> n;
    	int arr[n];
    	cout << "Enter elements: ";
    	for (int i = 0; i < n; i++) 
		cin >> arr[i];
    	ssort(arr, n);
    	cout << "Sorted list: ";
    	for (int i = 0; i < n; i++) 
		cout << arr[i] << " ";
    	cout << endl;
    	return 0;
}
            

def ssort(arr):
    	n = len(arr)
    	for i in range(n - 1):
		m=i;
        	for j in range(i + 1, n):
            		if arr[j] > arr[m]:
                		m=j;
		arr[i], arr[m] = arr[m], arr[i]

arr = eval(input("Enter list: "))
ssort(arr)
print("Sorted list:", arr)

Time Complexity:

  • Best Case: O[n2]
  • Worst Case: O[n2]
  • Average Case: O[n2]

Space Complexity:

  • O[1] (in-place, does not require additional space)

Used for:

  • Sorting when memory constraints are tight.

Applications:

  • Memory-Constrained Environments: Works without additional storage.
  • Simple Sorting Needs: Works well for small datasets.
  • Teaching Sorting Concepts: Helps understand swapping and selection operations.
